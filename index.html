ถ้าต้องการหน้า “รวมลิงก์” ที่ดูไม่เหมือนพอร์ทัลหน่วยงานเกินไป สไตล์บล็อกแบบ Blogger 2.0 จะช่วยให้หน้าเว็บดูเป็นโพสต์อ่านง่ายและคุ้นตา โดยยังคงความเป็นระเบียบด้วยวิดเจ็ตด้านข้างและกล่องรายการลิงก์

ในเนื้อหา สามารถอ้างอิงลิงก์ให้เป็นธรรมชาติ เช่น Data HK เพื่อให้บทความดูมีบริบท ไม่ใช่แค่ลิสต์ลิงก์ล้วน ๆ

แนวทางการใช้งานแบบโพสต์

ใช้โครงสร้างเหมือน “บทความ” ได้แก่ บทนำ → แนวทาง → รายการลิงก์ → ปุ่มทางออกหลัก และคงลิงก์ทั้งหมดไว้ตามเดิมเพื่อความต่อเนื่อง

หลักการจัดหน้าให้ดูดี

  • วางลิงก์สำคัญไว้ในเมนูด้านบนและวิดเจ็ตด้านข้าง
  • ทำรายการลิงก์ทั้งหมดเป็นกล่อง Directory ที่อ่านง่าย
  • มีปุ่ม CTA ชัดเจนเพื่อให้ผู้ใช้ไปปลายทางได้เร็ว
